# rust-snake

一个基于 Rust、`ratatui` 和 `crossterm` 的终端贪吃蛇项目，支持双蛇对战。

## 游戏截图

![](docs/screenshot.png)

## 当前功能

- 终端 TUI 界面
- 固定 tick 游戏循环（160ms）
- `WASD` 和方向键控制玩家移动
- `Enter` / `Space` / 方向键开始游戏
- `Space` 暂停/继续
- `r` 重新开始
- `q` 退出
- 玩家蛇吃到食物后增长
- 3 个 AI 敌蛇自动追逐食物
- 敌蛇撞死后会重生并保留分数
- 玩家撞墙、撞自己、或撞到敌蛇后游戏结束
- 分数显示（玩家分数 / 敌蛇总分）
- 按当前窗口大小初始化棋盘
- 调整窗口后按新尺寸自动开始新游戏
- 终端窗口过小时显示提示
- 支持 `-nocolor` 启动无色版本

## 项目结构

```text
src/
  main.rs     # 程序入口
  app.rs      # 终端生命周期、事件循环、输入处理
  game.rs     # 游戏状态与核心规则
  render.rs   # UI 布局与棋盘渲染
```

## 运行方式

确保本地已经安装 Rust 工具链，然后在项目根目录执行：

```bash
cargo run
```

启动无色版本：

```bash
cargo run -- -nocolor
```

## 测试

运行单元测试：

```bash
cargo test
```

## 操作说明

- `W` / `A` / `S` / `D` 或方向键：移动
- `Enter` / `Space` / 方向键：开始游戏
- `Space`：暂停或继续
- `r`：重新开始
- `q`：退出游戏

## 游戏说明

### 角色符号

| 角色 | 符号 | 颜色 |
|------|------|------|
| 玩家蛇头 | `@` | 绿色 |
| 玩家蛇身 | `o` | 绿色 |
| 敌蛇头 | `A` `B` `C` | 青色/黄色/红色等 |
| 敌蛇身 | `a` `b` `c` | 对应颜色 |
| 食物 | `*` | 红色 |
| 空格子 | `·` | 深灰色 |

### 游戏规则

- 玩家蛇(@)与 3 条 AI 敌蛇(A/B/C)同屏对战
- AI 敌蛇以 85% 概率追逐最近食物，15% 概率随机漫步（5-15 步）
- 双方都能吃食物(*)增长，食物同时存在 4 颗
- 食物数量会根据棋盘空间动态调整
- 玩家撞墙、撞自己、或撞到敌蛇则游戏结束
- 敌蛇撞死后会立即重生，分数保留
- 调整窗口大小会自动按新尺寸重新开始

### 状态面板

状态栏显示：Tick 数 | 玩家分数 | 敌蛇总分 | 敌蛇数量 | 玩家方向 | 所有敌蛇方向
