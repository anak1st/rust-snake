# Rust 贪吃蛇项目任务计划

## 当前进展总结

项目已经从空目录推进到了一个可运行的终端贪吃蛇原型，当前已完成：

- 已初始化 Cargo 项目并完成基础依赖接入
- 已接入 `ratatui`、`crossterm`、`rand`、`anyhow`
- 已完成终端进入、退出、备用屏切换和 raw mode 控制
- 已拆分为 `app`、`game`、`render` 三个主要模块
- 已实现固定 tick 的主循环
- 已支持 `WASD` 和方向键控制
- 已支持暂停、退出、重开
- 已实现蛇移动、吃食物增长、随机刷新食物
- 已实现撞墙和撞自己后进入 `Game Over`
- 已实现根据启动时窗口大小自适应棋盘尺寸
- 已实现窗口尺寸变化后按新尺寸重开一局
- 已补充核心逻辑单元测试并通过

## 目标

构建一个运行在终端中的 Rust 贪吃蛇游戏，优先保证：

- 核心玩法完整
- 输入响应稳定
- 终端渲染清晰
- 代码结构便于后续扩展

## 技术路线

- UI 方案：基于 TUI 终端界面实现
- 推荐框架：`ratatui`
- 事件与终端控制：`crossterm`
- 随机生成：`rand`
- 错误处理：`anyhow`
